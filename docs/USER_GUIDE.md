# Инструкция пользователя (кратко)

## 1. Запуск приложения

Откройте приложение в браузере после запуска контейнера:
http://localhost:8080

## 2. Роли

Доступны роли:

- **Специалист** — просмотр данных
- **Старший специалист** — дополнительно может:
  - редактировать значение измерения (клик по точке на графике)
  - добавлять/редактировать комментарий в истории состояний

Переключение роли выполняется через UI-переключатель (демо).

## 3. Навигация по регионам и теплицам

1. Слева отображается дерево **Регион → Теплица**.
2. В поле фильтра можно искать по названию региона или теплицы.
3. Клик по теплице открывает детальную карточку теплицы (графики, операции, история).

## 4. Просмотр измерений

1. В блоке управления выберите:
   - показатель: **Температура / Влажность / Кислотность**
   - период: **сутки / 7 дней / 30 дней / год**
2. Ниже отобразится график.

## 5. Редактирование измерения (только старший специалист)

1. Выберите диапазон **сутки** или **7 дней** (где точки видимы).
2. Кликните по точке на графике.
3. В модальном окне введите новое значение и сохраните.

## 6. История состояний

В таблице отображается история смены статусов:

- 0 — Норма
- 1 — Предупреждение
- 2 — Авария

Комментарий можно редактировать только в роли **старшего специалиста**.
