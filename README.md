# Greenhouse Monitoring UI

Прототип UI системы мониторинга и контроля состояния сельскохозяйственных теплиц (React + TypeScript + RTK Query + Recharts + SCSS + MSW).  
Фронтенд упакован в Docker-образ на базе **nginx** и готов к запуску.

---

## Run (Docker)

```bash
docker pull ghcr.io/alexandr-404/greenhouse-monitoring-app:latest
docker run --rm -p 8080:80 ghcr.io/alexandr-404/greenhouse-monitoring-app:latest
```

## Открыть в браузере:

```
http://localhost:8080
```
